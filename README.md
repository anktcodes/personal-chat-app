# Private Chat App

A simple, password-protected chat application for you and your friends to communicate in real-time.

## Features

- **Password Protection** - Secure login system to keep your conversations private
- **Real-time Messaging** - Send and receive messages instantly
- **Poke Feature** - Get your friends' attention with a quick poke notification
- **Clean Interface** - Simple, distraction-free chat experience
- **Username Display** - See who's saying what in the conversation

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, or Edge)
- A local web server or hosting service

### Installation

1. Download all three files:
   - `index.html` (main chat interface)
   - `style.css` (styling)
   - `script.js` (chat functionality)

2. Place all files in the same folder

3. Open `index.html` in your web browser, or host the files on a web server for shared access

### Usage

1. **Login**: Enter your username and the shared password
2. **Send Messages**: Type in the message box and click "Send" or press Enter
3. **Poke Friends**: Click the "Poke" button to send a notification
4. **Logout**: Click "Logout" when you're done chatting

## Configuration

To set up the app for your friend group:

1. Open `script.js` in a text editor
2. Update the password in the login verification function
3. Customize any other settings like usernames or styling as needed

## Technical Details

- Built with vanilla HTML, CSS, and JavaScript
- No external dependencies required
- Uses local storage for message persistence (optional)
- Lightweight and fast

## Notes

- This is a basic implementation designed for trusted friends
- For production use, consider adding proper backend authentication and database storage
- Messages may be stored locally depending on your implementation

## Support

If you run into any issues, check that:
- All three files are in the same directory
- JavaScript is enabled in your browser
- You're using the correct password
